# inVision U

**inVision U** — веб-платформа для подачи, обработки и оценки заявок в образовательную программу.  
Проект включает лендинг, многоэтапную анкету кандидата, голосовые ответы с транскрибацией, AI-анализ эссе и интервью, а также внутреннюю панель комиссии для просмотра и отбора участников.

## Возможности

- Лендинг сайта программы
- Многошаговая форма подачи заявки
- Сохранение данных кандидата
- Голосовые ответы с транскрибацией
- AI-анализ анкет, эссе и интервью
- Оценка кандидатов по нескольким критериям
- Вывод объяснений и причин выставленных оценок
- Панель комиссии для просмотра, фильтрации и удаления заявок
- Полностью русскоязычный интерфейс

## Технологии

- **Next.js**
- **React**
- **TypeScript**
- **Supabase**
- **OpenAI API**
- **Tailwind CSS**

## Структура проекта

- `app/` — страницы и API routes
- `components/` — переиспользуемые UI-компоненты
- `lib/` — вспомогательная логика, работа с Supabase, AI и авторизацией
- `public/` — статические файлы
- `styles/` — глобальные стили

## Переменные окружения

Создай файл `.env.local` и добавь в него:

```env
NEXT_PUBLIC_SUPABASE_URL=
SUPABASE_SERVICE_ROLE_KEY=
OPENAI_API_KEY=
COMMITTEE_PASSWORD=
```

## Установка и запуск

```bash
npm install
npm run dev
```

После запуска проект будет доступен локально по адресу:

```bash
http://localhost:3000
```

Если порт занят, Next.js автоматически запустит проект на другом, например `3001`.

## Основные разделы

### 1. Лендинг
Главная страница программы с описанием этапов, преимуществ и призывом к подаче заявки.

### 2. Форма кандидата
Многоэтапная анкета, в которой пользователь:
- заполняет личные данные
- отвечает на вопросы
- записывает голосовые ответы
- отправляет заявку на рассмотрение

### 3. AI-анализ
После отправки заявки система анализирует ответы кандидата и формирует:
- баллы по критериям
- сильные стороны
- зоны риска
- объяснение оценки
- общий вывод по кандидату

### 4. Панель комиссии
Внутренний кабинет для просмотра и оценки заявок:
- список кандидатов
- детальная карточка заявки
- удаление заявок
- просмотр AI-анализа
- принятие решений по кандидатам

## Назначение проекта

Цель проекта — упростить процесс отбора кандидатов в **inVision U**, сделать его более системным, прозрачным и удобным как для участников, так и для комиссии.

## Статус

Проект находится в активной доработке и может обновляться по мере развития программы и критериев отбора.
